# Sprout Journal - Ecosistema per Micro-ortaggi

Sprout Journal è una Web App premium progettata per la gestione e il monitoraggio dei lotti di micro-ortaggi. Offre un'interfaccia moderna, un timer intelligente per le irrigazioni e un sistema di monitoraggio della crescita.

## Caratteristiche principali

- **Monitoraggio Lotti**: Gestisci la crescita di ceci e lenticchie con timeline precise.
- **Timer Intelligente**: Notifiche per le irrigazioni con supporto per il tempo scaduto (overdue).
- **Design Premium**: Interfaccia minimalista e animata con icone che si espandono.
- **Supporto Offline**: I dati vengono salvati localmente per permettere l'uso anche senza connessione.
- **PWA Ready**: Installabile su smartphone come un'app nativa.

## Stack Tecnologico

- **Frontend**: React + TypeScript + Tailwind CSS
- **Animazioni**: Framer Motion
- **Database & Auth**: Firebase (Firestore & Authentication)

## Configurazione

1. Clona il repository.
2. Esegui `npm install`.
3. Crea un file `.env` basato su `.env.example` con le tue chiavi Firebase.
4. Esegui `npm run dev` per avviare l'ambiente di sviluppo.
